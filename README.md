# 👕 Amazon T-Shirt Niche Analyzer

A Python tool for analyzing Amazon T-shirt listings, identifying high-potential niches, and filtering copyright/trademarked content. This tool helps sellers uncover profitable product opportunities using real marketplace signals such as BSR, reviews, ratings, and keywords.

---

## 📌 Features

- 🔍 **CSV Analyzer** for Amazon T-shirt product data.
- 📊 **Winning Score Calculation** using BSR, ratings, reviews, and keywords.
- 🚫 **Copyright/Trademark Filter** using a large blocklist of common IP violations.
- 🧠 **Niche Classification** into categories like `family`, `seasonal`, `funny`, `sports`, etc.
- 🏆 **Top Niche Reporting** with market opportunity scores.
- 💡 **Strategic Recommendations** based on data-driven insights.
